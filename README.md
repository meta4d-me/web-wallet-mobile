# h5-web3-wallet-vue3-webpack-js-v1.1

基于 Vue3 + webpack5 + js 的h5-web3-wallet应用

### 环境要求
    node 16.14.0+

### 依赖包如下：
    "@coinbase/wallet-sdk": "^3.4.0",
    "@walletconnect/web3-provider": "^1.8.0",
    "axios": "^1.3.4",
    "ethers": "^5.6.8",
    "ipfs-http-client": "^56.0.3",
    "process": "^0.11.10",
    "jsrsasign": "^10.7.0",
    "mitt": "^3.0.0",
    "vant": "^4.1.0",
    "vue": "^3.2.47",
    "vue-router": "^4.1.6",
    "web3": "^1.9.0",
    "web3modal": "^1.9.8"

### 安装命令
    npm install

### 启动命令
    npm run dev

### 打包命令
    npm run build-test
    npm run build-uat
    npm run build-prod
